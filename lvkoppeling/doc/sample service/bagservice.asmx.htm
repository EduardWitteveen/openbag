BAGService


Service voor het ophalen van basisgegevens: pand, perceel, adres. Let op: daar waar een ADRES wordt opgehaald wordt eigenlijk een VERBLIJFSOBJECT bedoeld. Het Id van een adres is het id van het verblijfsobject - maar het teruggegeven object wordt adres genoemd omdat 'verblijfsobject' en vakterm is, en 'adres' een term is die beter in de belevingswereld van een burger past.

The following operations are supported. For a formal definition, please review the Service Description.

  • ZoekAdres
    Zoek adres op aanduiding

  • ZoekAdresFotos
    Zoek (WOZ) foto's bij een adres

  • ZoekAdresInCirkel
    Zoek adres binnen cirkel gedefinieerd door (Y1,Y1) en straal - coördinaten in RD stelsel

  • ZoekAdresInExtent
    Zoek adres binnen rechthoek gedefinieerd door (X1,Y1)-(X2,Y2)- coördinaten in RD stelsel

  • ZoekAdresOpOpenbareruimte
    Zoek adres openbare ruimte (= straat) id en huisnummer/letter/etc

  • ZoekAdresOpPand
    Zoek adres op id van een pand

  • ZoekAdresOpPerceel
    Zoek adres op kadstrale perceel aanduiding

  • ZoekAdresOpPostcode
    Zoek adres op postcode

  • ZoekGemeentelijkeMonumentenInPolygoon
    Zoek gemeentelijke monumenten by polygon (x1,y1 x2,y2 x3,y3 ...)

  • ZoekOpenbareruimteOpNaam
    Zoek openbare ruimte (= straat) op naam en eventueel woonplaats/straat

  • ZoekPandInCirkel
    Zoek perceel dat (deels) valt binnen een cirkel gedefinieerd door (Y1,Y1) en straal - coördinaten in RD

  • ZoekPandInExtent
    Zoek pand dat (deels) valt binnen rechthoek gedefinieerd door (X1,Y1)-(X2,Y2)- coördinaten in RD stelsel

  • ZoekPandOpAdres
    Zoek pand op id van het verblijfsobject (adres) id

  • ZoekPandOpNaam
    Zoek pand op naam (bijvoorbeeld "bibliotheek")

  • ZoekPandOpPerceel
    Zoek alle panden op een perceel

  • ZoekPandOpPunt
    Zoek pand op punt gedefinieerd door (X,Y)- coördinaten in RD stelsel

  • ZoekPerceelInCirkel
    Zoek perceel dat (deels) valt binnen een cirkel gedefinieerd door (Y1,Y1) en straal - coördinaten in RD stelsel

  • ZoekPerceelInExtent
    Zoek perceel dat (deels) valt binnen rechthoek gedefinieerd door (X1,Y1)-(X2,Y2)- coördinaten in RD stelsel

  • ZoekPerceelOpAanduiding
    Zoek perceel op kadastrale aanduiding

  • ZoekPerceelOpAdres
    Zoek perceel op verblijfsobject (adres) id

  • ZoekPerceelOpPand
    Zoek perceel waarop pand met opgegeven id ligt

  • ZoekPerceelOpPunt
    Zoek perceel op punt gedefinieerd door (X,Y)- coördinaten in RD stelsel

  • ZoekRijksMonumentenInPolygoon
    Zoek gemeentelijke monumenten by polygon (x1,y1 x2,y2 x3,y3 ...)
